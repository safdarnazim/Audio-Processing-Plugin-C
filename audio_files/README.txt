This folder contains the audio files used for testing the Audio Processing Plugin.

Included files:
- bgm.wav                : Background music sample
- vocals.wav             : Voice recording sample
- noise_audio.wav        : Sample noise file for testing denoising

Output files generated by the program will also appear here:
- output.wav
- output_lpf.wav
- output_hpf.wav
- output_denoise.wav
- output_pitch_shift.wav
- output_psb.wav   (playback speed modified file)

You may replace the default files with your own WAV files.
Files must be uncompressed PCM WAV format.
